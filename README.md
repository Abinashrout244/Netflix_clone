# Netflix Clone (React + Vite)

A Netflix-inspired movie streaming UI built using **React + Vite**, powered by **TMDB (The Movie Database) API**.  
This project focuses on modern UI, fast performance, and clean component architecture.

---

## Tech Stack

- React – UI Library
- Vite – Fast build tool
- Tailwind CSS – Styling
- TMDB API – Movie & TV data
- Firebase (optional) – Authentication
- React Router – Navigation

---

## Features

- Browse Trending, Popular & Top Rated Movies
- Search Movies & TV Shows
- Dynamic Banners & Rows (Netflix style)
- Fully Responsive Design
- Movie Details with Rating & Overview
- Authentication (Login / Signup)
- Fast Loading with Vite
